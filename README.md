## Features

* Can point to magnetic north as well as true north
* Shows exact degrees and cardinal direction
* Displays sensor status
* Haptic feedback
* Follow system, Light and Dark theme
* Automatic Dark theme
* Localization
* No advertisements
* No trackers
* No annoyance
